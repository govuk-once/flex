# @flex/sdk-service-gw

Declarative domain configuration for the FLEX Platform Service GWs.

---

## Commands

Run these from the repository root:

| Command                                   | Description    |
| ----------------------------------------- | -------------- |
| `pnpm --filter @flex/sdk-service-gw lint` | Lint files     |
| `pnpm --filter @flex/sdk-service-gw test` | Run tests      |
| `pnpm --filter @flex/sdk-service-gw tsc`  | Run type check |

Alternatively, run `pnpm <command>` from within `libs/sdk-service-gw/`.
